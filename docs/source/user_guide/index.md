# User Guide

All schema extend the base {py:class}`XarraySchema` class and are functionally equivalent. But it makes sense to separate them logically, based on the way they are composed. It is helpful to distinguish "container" schema from "component" schema. Simply put, *container* schema compose *component* schema.
